## Simple Interest Calculator

A calculator that calculates simple interest given principal, annual rate of interest and time period in years.
The calculator uses the inputs of and calculates the interest on the principal using a decimal fraction for percent. The
Output result does not include the principal amount, only the simple interest amount.

```
Input:
   p, principal amount
   t, time period in years
   r, annual rate of interest
Output
   simple interest = p*t*r
```
